Estudos em HTML5 e CSS3

Este repositório reúne exercícios, desafios e pequenos projetos desenvolvidos durante minha trilha de estudos em HTML5 e CSS3, com base no curso do Curso em Vídeo.

O objetivo deste repositório é documentar meu processo de aprendizado, evolução técnica e prática contínua em desenvolvimento web.

---

Objetivo do Repositório

- Praticar fundamentos de HTML semântico
- Aplicar conceitos básicos de CSS
- Resolver desafios propostos durante o curso
- Consolidar lógica de estruturação de páginas
- Registrar minha evolução como estudante de desenvolvimento web

---

Conteúdo

O repositório contém:

- Exercícios práticos
- Desafios propostos no curso
- Pequenos sites e páginas estáticas
- Exemplos de uso de HTML e CSS

Observação:  
Os projetos aqui presentes são exercícios de aprendizado, alguns de nível básico, e não representam projetos finais de portfólio.

---

Tecnologias Utilizadas

- HTML5
- CSS3
- Git & GitHub
