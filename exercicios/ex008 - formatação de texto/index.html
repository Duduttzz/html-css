formatação de texto

itálico - enfase

para deixar o texto em itálico usamos <em>

negrito - destaque

para deixar o texto em negrito usamos <strong>

marcar texto

para deixar um texto marcado usamos <mark>

texto pequeno

para deixar um texto pequenininho usamos <small>

texto excluido (que deve ser lido mas não considerado)

para fazer isso usamos <del>

texto sublinhado

para deixar o texto sublinhado usamos <ins>

texto sobrescrito (elevado)

para deixar o texto sobrescrito usamos <sup>

texto subscrito

para deixar o texto sunscrito usamos <sub>

código-fonte (fonte mono-espaçada)

para deixar o texto mono-espaçado usamos <code>

considerar as identações e quebras de linhas digitadas no codigo html

para
    fazer
    isso usamos:
    <pre>

citação simples (inserir aspas)

para fazer uma citação usamos <q>

citação completa

para fazer isso:

usamos <blockquote>

podemos junta-lo com o cite="url" para referenciar qual foi o site de onde tirou a citação

abreviação

para inserir o sing da abreviação usamos <abbr>

texto invertido

para escrever assim usamos <bdo>
